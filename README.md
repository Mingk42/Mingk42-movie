# movie

### install
```bash
# main
$ pip install git+https://github.com/Mingk42/-Mingk42-movie.git

# branch
$ pip install git+https://github.com/Mingk42/-Mingk42-movie.git@<branch_name>
```

### start dev
```bash
$ git clone <url>
$ cd <dir>
$ pdm venv create            # optional
$ source .venv/bin/activate
```
